Building System

A project to study the system and structure of architecture

  1. Home
  2. Architecture
  3. Building System

Based on an existing office in St. Louis, this project focuses on analyzing building components and structure of the building before and after renovation. It enables me to learn more and materials, supporting structures and how they are connected.
